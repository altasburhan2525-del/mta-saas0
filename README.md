# MT Altaş Hesaplayıcı

`app.mtaltasinsaat.com` için temiz başlangıç.

Bu proje, MT Altaş Hesaplayıcı sistemini ileride SaaS, bayi paneli ve web sitelerine gömülebilir embed yapısına taşımak için hazırlanmıştır.

## Kurulum

```bash
npm install
npm run dev
```

## Yayın

Vercel üzerinden yayınlanır. Hedef domain:

```text
https://app.mtaltasinsaat.com
```

## Marka adı

Dışarıya görünen isim: **MT Altaş Hesaplayıcı**

## İlk hedef

- Temiz ana sayfa
- Embed yayınlama mantığı
- API temel kontrol noktası
- Bayi ve SaaS sistemine hazır mimari
